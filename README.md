# Plant Store 🌿

A modern and responsive Plant Store web application built using React and Vite. The website provides a clean UI for browsing plants, exploring categories, and viewing plant product cards with a visually appealing design.

## 🚀 Live Demo

[https://plant-store-eight-taupe.vercel.app/](https://plant-store-eight-taupe.vercel.app/)

---

# ✨ Features

* Responsive modern UI
* Beautiful plant product cards
* Fast performance using Vite
* Reusable React components
* Tailwind CSS styling
* Smooth and clean user experience
* Optimized frontend deployment on Vercel

# ⚙️ Installation & Setup

## 1. Clone the repository

```bash
git clone https://github.com/Disha236/Plant-Store.git
```

## 2. Navigate into the project folder

```bash
cd Plant-Store
```

## 3. Install dependencies

```bash
npm install
```

## 4. Start development server

```bash
npm run dev
```

The app will run on:

```bash
http://localhost:5173
```

---

# 🏗️ Build for Production

```bash
npm run build
```

---

# 🌐 Deployment

This project is deployed using Vercel.

## Deploy on Vercel

1. Push the code to GitHub
2. Import the repository into Vercel
3. Configure build settings:

```bash
Build Command: npm run build
Output Directory: dist
```

4. Click Deploy
